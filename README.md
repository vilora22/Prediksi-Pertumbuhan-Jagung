# Prediksi-Pertumbuhan-Jagung

Proyek ini bertujuan untuk memprediksi laju pertumbuhan jagung berdasarkan faktor lingkungan menggunakan algoritma SVR, Linear Regression, dan Random Forest.

# Informasi Dataset
Dataset ini berisi informasi tentang variabel lingkungan dan tanah yang memengaruhi pertumbuhan tanaman jagung. Data dikumpulkan dari tahun 2010 hingga 2019 dengan tujuan untuk memprediksi dan mengoptimalkan praktik pengelolaan guna meningkatkan produktivitas jagung.

Variabel

Suhu
Suhu lingkungan dalam derajat Celcius.
Tipe Data: Numerik (float)
Rentang Nilai: 10°C hingga 35°C
Pentingnya: Suhu secara langsung memengaruhi laju pertumbuhan dan perkembangan jagung. Suhu dalam kisaran optimal mendorong pertumbuhan tanaman yang ideal.

Kelembapan
Kelembapan relatif dalam persentase.
Tipe Data: Numerik (float)
Rentang Nilai: 30% hingga 90%
Pentingnya: Kelembapan relatif sangat penting untuk transpirasi dan pertukaran gas pada daun jagung, yang memengaruhi penyerapan nutrisi dan air.

Kelembapan Tanah:
Kelembapan tanah dalam persentase.
Tipe Data: Numerik (float)
Rentang Nilai: 10% hingga 40%
Pentingnya: Kelembapan tanah sangat penting untuk ketersediaan air bagi akar jagung, terutama selama tahap kritis perkecambahan dan pertumbuhan vegetatif.

Pertumbuhan
Jagung dalam sentimeter.
Tipe Data: Numerik (float)
Rentang Nilai: Bergantung pada kondisi lingkungan dan tanah.
Pentingnya: Pengukuran langsung kesehatan dan perkembangan tanaman, dipengaruhi oleh variabel lingkungan dan tanah. Ini adalah variabel target untuk model regresi.
